# Client Handover (What the client receives)

## 1) What you get
- A mobile-first storefront link (opens on any phone)
- Products + prices + photos
- WhatsApp / Messenger ordering

## 2) How customers order
1) Customer opens the link
2) Taps a product
3) Taps WhatsApp or Messenger
4) A message is generated automatically
5) You confirm details and delivery

## 3) What you (client) can change later
If you want changes (new products / new prices / new photos), send us:
- product name
- price
- photo(s)
- short description

We can update quickly.

## 4) Support rules (simple)
- Minor text changes: same day
- New products: depends on quantity
- Bugs (rare): fixed ASAP
